# Zachary Lorenzo Sulit — Data Science Portfolio

A personal portfolio website for Zachary Lorenzo Sulit, a Data Science student focused on machine learning, analytics, visualization, and data storytelling.

## Project Overview

This is a static website built with HTML, CSS, and JavaScript. It presents Zachary's profile, featured data science projects, technical skills, resume, and contact links in an interactive portfolio experience.

## Features

- Intro landing screen with branded logo animation
- Responsive navigation with smooth section scrolling
- Interactive Three.js hero background
- About section with personal branding note
- Project cards featuring machine learning, F1 analytics, dashboards, and DOST-PHIVOLCS work
- Skills and technology stack section
- Resume/timeline section with downloadable CV
- Contact form powered by Formspree
- Custom cursor, reveal animations, and responsive styling

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Three.js via CDN
- Google Fonts
- Formspree contact form

## Project Structure

## How to Run

Open `index.html` directly in a browser, or serve the folder with any local static server.
